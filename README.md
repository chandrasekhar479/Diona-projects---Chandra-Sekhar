# 🏥 Medical & Travel Expense Request

This is a frontend web application developed using **HTML, CSS, and JavaScript**.  
The application allows employees to submit medical and travel expense reimbursement requests online.  
It includes modules for prescription drugs, medical supplies, parking charges, mileage, and transportation expenses.  
JavaScript is used for form validation and improving the user experience.  
The application features a clean, responsive, and user-friendly interface.  
This project helps reduce paperwork and simplifies the expense claim process.  
It is suitable for learning frontend development and form-based web applications.

# 📋 Worker Progress Report

This is a frontend web application developed using **HTML, CSS, and JavaScript**.  
The application allows employees to submit their recovery and return-to-work progress online.  
Users can update their work status, pain level, medical treatment, and recovery details through an easy-to-use form.  
JavaScript is used for form validation and improving the user experience.  
The application features a clean, responsive, and user-friendly interface.  
This project helps reduce paperwork and simplifies the worker progress reporting process.  
It is a good example of a frontend form-based web application.
